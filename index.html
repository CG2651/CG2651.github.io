School Projects

Galactic Pizza Party

Galactic Pizza Party is a Unity VR game, built for Quest 2, where you deliver pizzas to aliens by swinging around the city. The video shows an older concept where the player originally deliver packages. I will update the project soon.

Alien model was made using Adobe Photoshop and Autodesk Maya.

Abattoir

Abattoir is a 2D side-scrolling game that uses a webcam, Victorian calling cards, and an optional Victorian trivia sheet to unlock doors in a dark castle. This project was created with 4 other students, and made in Unity with Vuforia for AR Image Tracking.

My contributions to the project was providing code for player movement and falling platforms, animating the enemies, and testing the AR Image Tracking.

Mantis

Mantis is a 2D side-scrolling game where you play as a mantis fighter looking for their bag that was stolen from sharks. This project was my first introduction to Unity and game development.

Characters and animated cutscene was made using Adobe Animate. Other visual assets were made with paint.NET.

Personal Projects

In The Bayou

In The Bayou is a supernatural, horror visual novel made for the Bad Art Visual Novel Jam. This was made in Renpy with 2 other developers. The theme for the jam was "Transformation".

My contributions to the project was programming all the dialogue and player choices.

Plants vs Bugs

Plants vs Bugs is a 2D top-down game inspired by Plants vs Zombies and Bloons TD. This was made in Unity with 4 other developers and 4 composers for the Game/Music Jam 2022. The theme for the jam was "Growth".

My contributions to the project was creating all the visual assets, testing, and editing the trailer.
